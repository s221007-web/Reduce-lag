# Reduce Lag 🚀

Hello! Welcome to my project. I am a student developer learning how to code, and this is my tool to help reduce system lag on heavy or chaotic websites. 

Since I am still learning, the code might not be perfect, but I am working hard to make it better!

---

## 🚀 How to Use It

You can use this tool in **two different ways**, depending on your preference:

### Option 1: As a Bookmarklet
1. Copy the raw code inside **`bookmarklet.js`**.
2. Create a new bookmark in your browser bar.
3. Paste the code into the **URL field** of the bookmark.
4. Click the bookmark on any webpage to instantly freeze it!

### Option 2: As a Userscript
1. Install a userscript manager extension like **Tampermonkey**, **Greasemonkey**, or **Violentmonkey**.
2. Create a new script and paste the contents of **`userscript.js`** into it.
3. Save the script. It will now stand ready to trigger via your shortcut key on any website.
